# Copywriting: Voice & Tone Guidelines

When generating text, adhere to the Ganesh Residency brand persona.

## Brand Personality Keywords
* Professional but welcoming
* Luxury and Premium
* Direct and efficient
* Tech-savvy but jargon-free
* Empathetic and Hospitable

## Grammar & Mechanics rules
* **Headings:** Use Title Case for main headings (H1, H2). Use sentence case for subheadings (H3+).
* **Punctuation:** Avoid exclamation points (!) in standard interface copy to maintain a premium feel. Use periods for complete sentences.
* **Clarity:** Prefer active voice over passive voice. Keep sentences concise. Communicate transparency and trust.

## Terminology Guide

| Do Not Use | Use Instead |
| :--- | :--- |
| "Utilize" | "Use" |
| "In order to..." | "To..." |
| "Cheap" | "Value" or "Affordable" |
| "Customer" | "Guest" |
| "Purchase" | "Book" |
